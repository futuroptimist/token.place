# token.place desktop (legacy Electron prototype)

> **Status: deprecated prototype**
>
> The current `desktop/` implementation is a legacy Electron path and is **not**
> the forward-looking desktop direction for token.place.
>
> The recommended architecture is now a Tauri-based desktop client focused on
> local llama.cpp inference, local encryption, and encrypted forwarding through
> `relay.py`.
>
> See the canonical design doc: [`docs/design/tauri_desktop_client.md`](../docs/design/tauri_desktop_client.md).
>
> The Electron prototype is being retained temporarily for historical context and
> short-term reference. Retirement/removal will happen in a later migration task.
